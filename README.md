# Chatbot Front - SigmaMed

Interface moderna construída com Next.js para interação com o chatbot SigmaMed.

## 🚀 Como Iniciar

1. Instale as dependências:
   ```bash
   npm install
   ```

2. Configure o ambiente:
   Crie um arquivo `.env` baseado no `.env.example` (ou use os valores padrão para local):
   ```env
   NEXT_PUBLIC_API_URL=/api
   CHATBOT_API_URL=http://localhost:3001/api
   ```

3. Inicie o servidor de desenvolvimento:
   ```bash
   npm run dev
   ```

4. Acesse: [http://localhost:3000](http://localhost:3000)
